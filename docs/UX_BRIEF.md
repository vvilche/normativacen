# Brief UX/UI Interactivo

Ejecuta el asistente desde la terminal para capturar el contexto antes de diseñar nuevas pantallas.

```bash
npm run ux:brief
```

El script hace preguntas sobre:

- Audiencia principal
- Objetivo primario/secundario
- Tono visual deseado
- Mensaje del hero
- Métricas clave
- CTAs
- Recursos a destacar
- Restricciones técnicas/branding

Al finalizar imprime un resumen listo para compartir con diseño o producto.
