# Journeys Plataforma IA NormativaCEN

## Usuario Exploratorio (Modo Guía)
- **Perfil**: Coordinado nuevo o personal administrativo que necesita fundamentos.
- **Objetivos**:
  - Comprender conceptos clave (SITR, PMUS, EDAC, etc.).
  - Obtener sugerencias de siguientes preguntas / capacitación.
  - Recibir checklists simples orientados a cumplimiento básico.
- **Requisitos UX**:
  - Copys educativos con tono accesible.
  - Bloques "Aprende más" con 2-3 preguntas sugeridas y enlaces a capacitación.
  - CTA hacia módulo de entrenamiento.
- **Salidas esperadas**:
  - Resumen básico + mini checklist.
  - Sugerencias de "Capacitación recomendada".
  - Registro del prompt para analytics.

## Usuario Experto (Modo Operativo)
- **Perfil**: Ingenieros OT/Compliance con incidentes específicos.
- **Objetivos**:
  - Obtener diagnóstico accionable y consecuencias normativas.
  - Recibir checklist avanzado con responsables/plazos.
  - Ver métricas de riesgo y tiempos del orquestador.
- **Requisitos UX**:
  - Callouts técnicos, tablas detalladas, timelines.
  - Chips con tiempos RAG/LLM/Auditor.
  - Botón rápido a PDF/Dossier para auditorías.
- **Salidas esperadas**:
  - Dossier técnico completo.
  - Plan de acción con prioridades.
  - Registro completo para trazabilidad y analítica.
