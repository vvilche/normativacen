# Prompt Guides

## Set Básico (Modo Guía)
1. ¿Qué es el SITR y cómo afecta a mi compañía?
2. ¿Cuáles son los pasos iniciales para actualizar el PMUS 2026?
3. ¿Qué evidencias EDAC debo reunir antes de una auditoría?
4. ¿Cómo funciona el CDC del CEN y qué debo reportar?

Para cada respuesta:
- Resumen educativo.
- Lista de "Preguntas sugeridas" (2-3).
- CTA hacia Capacitación.

## Set Experto (Modo Operativo)
1. "Mi GPS no genera timestamp. ¿Qué consecuencias tiene?"
2. "El enlace SITR excede 500ms. ¿Cómo recupero cumplimento?"
3. "Mi BESS debe cumplir CIP-013 y FFR<200ms. Plan inmediato."
4. "Tengo desfasajes PMU con C37.118. ¿Qué presenta el CEN?"

Para cada respuesta:
- Diagnóstico técnico + impacto regulatorio.
- Checklist avanzado con responsables/plazos.
- Métricas derivadas y tiempos de orquestador.
